# Kovancilarmatematik Backend

Bu proje Kovancilarmatematik uygulamasının backend kısmıdır.

## Teknolojiler

- Node.js
- Express.js
- Prisma ORM
- PostgreSQL
- TypeScript

## Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Veritabanını ayarlayın:
```bash
npx prisma migrate dev
```

3. Uygulamayı başlatın:
```bash
npm run dev
```

## API Endpoints

API dokümantasyonu için Postman koleksiyonunu kullanabilirsiniz.
